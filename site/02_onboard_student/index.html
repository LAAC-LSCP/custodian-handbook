How to Onboard a New Student¶

This guide is for supervisors and lab managers setting up access for a new student collaborator. Complete each step in order — some steps require waiting for account approvals before the next can begin.

Step 1 — Institutional Prerequisites¶

  1. Ethics training — The student must complete their institution's mandatory research ethics training (e.g. TCPS2 CORE, CITI Program, or equivalent). Send the supervisor a copy of the completion certificate.
  2. Sign the Data Sharing Agreement — Provide the student with the current version from the project website and ensure it is returned signed before granting any data access.
  3. Add to the team list — Email the project coordinator with the student's full name, institution, supervisor name, and anticipated end date so they can be added to the collaborators registry.

Step 2 — Databrary Registration¶

Databrary is the primary platform for storing and sharing video data within ExELang.

  1. Go to https://databrary.org and click Register.
  2. Select Affiliate as the account type (not Principal Investigator).
  3. When prompted for a sponsor, enter the PI's name and institutional email address. The PI will receive an authorisation request by email.
  4. The PI approves the request from their Databrary account under Manage Affiliates.
  5. Once approved, the student will receive a confirmation email. Ask them to forward it to the project coordinator.
  6. The coordinator will then grant access to the ExELang Databrary volume.

Note: Databrary approval can take 2–5 business days. Plan onboarding accordingly.

Step 3 — Server & Tools Access¶

  1. Request a server account — Email the technical lead with the subject line [ExELang] New user account request. Include the student's full name, institution, and expected start date.
  2. SSH key setup — Ask the student to generate an SSH key pair and send their public key to the technical lead. Instructions are in Server_Access_Guide.pdf on the shared drive.
  3. REDCap access — Submit a REDCap access request through the institutional portal. Select the ExELang project and choose the Data Entry role for most students.

Step 4 — Orientation Resources¶

Once accounts are set up, share the following with the new student:

  • This training guide (the document you are reading now)
  • The ExELang project website: https://exelang.org
  • The shared drive folder: ExELang > Onboarding > New Collaborator Starter Pack
  • Schedule an introductory call with the PI or a senior team member within the first week
